# Arşiv — Eski Streamlit Uygulamaları

Bu klasördeki dosyalar **kullanımdan kaldırıldı**. Güncel uygulama:

```
streamlit run app.py
```

| Dosya | Açıklama |
|-------|----------|
| `app_hybrid.py` | LR + tür önceliği hibrit demo (slider etkileşimi için) |
| `app_old.py` | Eski regresyon dashboard (Bagging Regressor) |
| `app_new.py` | Sınıflandırma taslağı (hatalı feature engineering) |
| `app_streamlit.py` | Kısa regresyon prototipi |

Hibrit demo: `streamlit run archive/app_hybrid.py`
